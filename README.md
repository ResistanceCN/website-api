# website-api

## Introduction
[GraphQL](http://graphql.org/) API for [our website](https://github.com/ResistanceCN/website-react). Build with [Flask](https://github.com/pallets/flask).

## Requirements

* Python 3.7+
* MongoDB 4.0+
